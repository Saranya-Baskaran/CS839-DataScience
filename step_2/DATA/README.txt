source1.csv is the table that contains books extracted from amazon.com
source2.csv is the table that contains books extracted from goodreads.com

source1.csv contains 3387 tuples.
source2.csv contains 3001 tuples.

The following are the attributes of the schema -

1. Name - Name of the book. Type - String
2. Author - Author of the book. Type - String
3. Publisher - Publisher of the book. Type - String
4. Publishing_Date - Date when the book was published. Type - String of format month-day-year
5. Format - Format of the book (For example, hardcover, paperback, kindle). Type - String
6. Pages - Number of pages in the book. Type - Int
7. Rating - Rating of the book out of 5. Type - Float
